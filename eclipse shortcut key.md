
이클립스 단축키 :
===============
* md 문법 참고 : <https://gist.github.com/ihoneymon/652be052a0727ad59601>

- ctrl + space : 자동완성

- shift + enter : 바로 아랫줄로 엔터
- Ctrl + d : 행 삭제
- ctrl + alt + ↓ : 행 복사 (어떤 PC에선 잘 안됨...)
- alt + 상하 : 행 이동
- ctrl + shift + c : 주석 처리
- ctrl + shift + / : 블록 주석처리
- ctrl + / : 행 주석처리 (어떤 PC에선 잘 안됨...)
- alt + shift + y : 긴 코드 나란히 보기

- ctrl + m : 에디터 창 크게 보기
- ctrl + (+)or(-) : 에디터 글씨 크게, 작게
